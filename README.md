# Bhava

Boutique wellness micro-resort development in Costa Rica. Design-led, feminine-aesthetic, wellness-integrated hospitality at the 5–12 villa scale.

This repository hosts the Bhava venture analysis as a static site via GitHub Pages.

## Pages

- **Foundation** — Industry structural landscape. 10 structural forces, 5 root causes, 4 boundary conditions, 4 feedback loops.
- **Problem** — Causal analysis of the convergent challenge facing new boutique entrants serving long-haul wellness travelers.

The analysis is generated by [Venture Forge](https://github.com/CdrAppDev/venture-forge), an adversarial multi-agent venture-validation pipeline.
